rn-spinner
---

Number Spinner component for react-native.

## Install

```
npm install rn-spinner --save
```

## Usage

```
// Require
var Spinner = require('rn-spinner');

// Use
<Spinner max={10} min={2} onNumChange={(num)=>{console.log(num)}}
```

## Props

- max
- min
- onNumChange: get the number of the Spinner
